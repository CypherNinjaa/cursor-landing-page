# Dev Tool Landing Page - Cursor Clone

A desktop-first developer tool landing page inspired by the [Cursor website](https://cursor.com). This project recreates the visual design and structure of Cursor's landing page using only HTML and CSS.

## Project Overview

This assignment is part of the **Web Dev Cohort 2026** program. The goal is to achieve visual and structural accuracy, matching the original Cursor website as closely as possible.

## Sections Recreated

| #   | Section                         | Description                                                                                            |
| --- | ------------------------------- | ------------------------------------------------------------------------------------------------------ |
| 1   | **Top Navigation Bar**          | Fixed navbar with logo, nav links (Features, Enterprise, Pricing), and CTA buttons (Sign in, Download) |
| 2   | **Hero Section**                | Main headline, subtitle, download CTA button, and large product screenshot mockup                      |
| 3   | **Trusted By / Logos**          | Row of company logos showing trusted partners                                                          |
| 4   | **Feature Sections (3 blocks)** | Two-column layouts with alternating text/image positions for Agent, Tab, and Ecosystem features        |

| 6   | **Testimonials**                | Quote cards with testimonials from industry leaders                                                    |
| 7   | **Use Cases / Stories**         | Blog highlight cards with image and short text                                                         |
| 8   | **Changelog / Updates**         | List of recent updates with version numbers and dates                                                  |
| 9   | **Team / About**                | Large image section with company mission and CTA                                                       |
| 10  | **Final CTA**                   | Big heading with download button                                                                       |
| 11  | **Footer**                      | Multi-column links, company info, and social links                                                     |

## How to View

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. For best results, view on a desktop screen (1200px+ width)

## Features

- Semantic HTML5 structure
- CSS custom properties (variables) for consistent theming
- Clean and organized code with comments
- No JavaScript required
- No external CSS frameworks (no Tailwind)
- Desktop-first design
- Proper typography hierarchy
- Accurate color palette
- SVG icons for crisp graphics

## Constraints Met

- [x] HTML and CSS only
- [x] No JavaScript
- [x] No TailwindCSS
- [x] No animations or fancy effects
- [x] Desktop-only layout (no responsiveness required)
- [x] Brand assets from original website used where possible
- [x] Fonts, colors, and layout matched to original design
